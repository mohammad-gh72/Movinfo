## Movinfo app

این پروژه یک وبسایت است که در آن کاربران می توانند به جستجوی فیلم ها و خواندن اطلاعات مرتبط با آن بپردازند
این پروژه زمانی ساخته شد که من در حال یادگیری مباحث ابتدایی پیرامون ,ری اکت روتر و هوک یوزردیوسر و همچین ساخت کاستوم هوک ها بودم بودم
از این رو این پروژه محدود به استفاده از تکنولوژی های حول محور موارد گفته شده می باشد.

this project is about a website where users can search for movies and explore related information.
This project was created during my exploration of React Router fundamentals, UseReducer Hooks, and custom hooks. Therefore, it focuses specifically on utilizing these technologies.
